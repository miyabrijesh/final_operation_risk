## 🧠 Project Overview

This project is an **AI-powered decision support system** designed to analyze e-commerce returns and optimize warehouse operations.

It predicts:
- 🔍 Fraud Risk of a return  
- 📊 Processing Category (Low / Medium / High effort)  
- ⏱ Operational Impact (inspection & delay factors)  

The system combines:
- Machine Learning models  
- Real-world dataset analysis  
- Consumer behavior insights  

---

## 🎯 Objectives

- Detect fraudulent return requests  
- Optimize warehouse processing decisions  
- Reduce unnecessary inspections  
- Improve operational efficiency

## ⚡ Optimization Logic

| Fraud Risk | Action |
|----------|--------|
| Low | Fast-track processing |
| Medium | Manual inspection |
| High | Intensive inspection |

---

## 📈 Key Insights

- Fraud probability varies by return reason  
- High warehouse load increases delays  
- Inspection level impacts processing time  
- Risk-based routing improves efficiency
